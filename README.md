# radio-writing-tool
AI Co-Authoring tool for Radio Play

## Usage

### Basic HTML app

`npm install http-server -g`

`http-server` 

open `localhost:8080` in Safari (speechRec isn't working in edge)

### Node app with pm2

`npm install pm2 -g`
`npm install -g express`
